# Bunext

A Next.js 15 app with Tailwind CSS template.

## Usage (run locally)

Go to the `root` folder where `package.json` exists.

```bash
bun install
```

```bash
npm install
```

### Then

```bash
bun --bun run dev
```

```bash
bun run dev
```

```bash
npm run dev
```

> required `nodejs` or `bun` installed

## Commit/Push

Prepare Husky

```bash
npm run prepare
```

## Features

- Next.js 15 App Directory
- Radix UI Primitives
- Tailwind CSS
- Tailwind CSS class sorting, merging and linting.
